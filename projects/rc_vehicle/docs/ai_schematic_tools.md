# Инструменты для генерации схем подключения

## AI инструменты для создания схем

### 1. **ChatGPT / Claude / Gemini**
Можно использовать для генерации описания схемы, которое затем можно визуализировать.

**Промпт пример:**
```
Создай детальную схему подключения для проекта RC Vehicle:
- ESP32-S3 (GPIO17 TX, GPIO18 RX) подключен к RP2040 через UART
- RP2040 генерирует PWM на GPIO2 (ESC) и GPIO3 (Servo)
- RP2040 читает RC сигналы на GPIO4 и GPIO5
- MPU-6050 подключен к RP2040 через I2C (GPIO6 SDA, GPIO7 SCL)
- Питание: BEC 5V для ESP32, RP2040, Servo
- Все компоненты на общей земле

Создай ASCII схему и описание подключений.
```

### 2. **draw.io / diagrams.net** (Рекомендуется)
- **URL**: https://app.diagrams.net/
- **Особенности**: Бесплатный, работает в браузере, экспорт в PNG/SVG/PDF
- **Использование**: Откройте файл `wiring_diagram.drawio` в draw.io
- **AI функции**: Нет встроенного AI, но есть шаблоны и библиотеки компонентов

### 3. **KiCad** (Для печатных плат)
- **URL**: https://www.kicad.org/
- **Особенности**: Профессиональный инструмент для создания схем и печатных плат
- **AI функции**: Нет, но можно создать полноценную схему и разводку платы
- **Использование**: Создайте новую схему и добавьте компоненты вручную

### 4. **Fritzing**
- **URL**: https://fritzing.org/
- **Особенности**: Удобен для breadboard схем, есть библиотека компонентов
- **AI функции**: Нет
- **Использование**: Хорош для визуализации макетной платы

### 5. **EasyEDA**
- **URL**: https://easyeda.com/
- **Особенности**: Онлайн редактор схем и печатных плат
- **AI функции**: Нет, но есть автодорожки для печатных плат
- **Использование**: Создайте схему в браузере

### 6. **CircuitLab**
- **URL**: https://www.circuitlab.com/
- **Особенности**: Онлайн симулятор схем
- **AI функции**: Нет
- **Использование**: Хорош для проверки электрических параметров

## Специализированные AI инструменты

### 7. **GitHub Copilot / Cursor AI**
Можно использовать для генерации кода схемы в различных форматах (KiCad, SPICE, etc.)

### 8. **Perplexity AI / Phind**
Можно спросить: "Создай схему подключения для [описание проекта]" и получить детальное описание.

### 9. **AutoDesk Fusion 360** (с AI функциями)
- **URL**: https://www.autodesk.com/products/fusion-360/
- **Особенности**: 3D моделирование с возможностью создания схем
- **AI функции**: Ограниченные, но есть автогенерация некоторых элементов

## Генерация через AI (пример промпта)

Для ChatGPT/Claude:

```
Создай детальную электрическую схему подключения для проекта RC Vehicle со следующими компонентами:

1. ESP32-S3 Zero mini:
   - GPIO17: TX (UART)
   - GPIO18: RX (UART)
   - Питание: 5V от BEC

2. Raspberry Pi Pico (RP2040):
   - GPIO0: UART TX к ESP32
   - GPIO1: UART RX от ESP32
   - GPIO2: PWM выход для ESC (Throttle)
   - GPIO3: PWM выход для Servo (Steering)
   - GPIO4: PWM вход от RC приёмника CH1
   - GPIO5: PWM вход от RC приёмника CH2
   - GPIO6: I2C SDA к MPU-6050
   - GPIO7: I2C SCL к MPU-6050
   - Питание: 5V от BEC

3. MPU-6050 (IMU):
   - SDA: GPIO6
   - SCL: GPIO7
   - VCC: 3.3V
   - GND: общая земля

4. RC Приёмник (Himoto 4CH):
   - CH1: GPIO4 (через делитель напряжения если 5V)
   - CH2: GPIO5 (через делитель напряжения если 5V)
   - Питание: 5V от BEC

5. ESC и Servo:
   - ESC Signal: GPIO2
   - Servo Signal: GPIO3
   - Питание Servo: 5V от BEC

6. BEC (Battery Eliminator Circuit):
   - Вход: Батарея 2S LiPo (7.4V)
   - Выход: 5V для всех компонентов

Создай:
1. ASCII схему подключений
2. Таблицу подключений
3. Список необходимых компонентов (резисторы, конденсаторы)
4. Рекомендации по безопасности
```
