# Настройка коэффициентов фильтров и стабилизации через веб-интерфейс

## Подключение к машинке

1. Включите машинку (ESP32-S3).
2. На телефоне или ноутбуке подключитесь к Wi-Fi сети машинки (имя сети отображается в UART-логе при старте, по умолчанию `RC_Vehicle_XXXX`).
3. Откройте браузер и перейдите по адресу `http://192.168.4.1/`.
4. Дождитесь, пока статус WebSocket покажет **Подключено** (зелёный).

## Обзор интерфейса

Веб-интерфейс содержит следующие панели (сверху вниз):

| Панель | Назначение |
|--------|-----------|
| Статус | Подключение WebSocket и контроллера |
| Wi-Fi клиент | Подключение машинки к домашней сети |
| Управление | Слайдеры газа и руля |
| Калибровка IMU | Калибровка гироскопа и акселерометра |
| **Стабилизация** | **Настройка режимов и коэффициентов** |
| Лог телеметрии | Просмотр и скачивание данных |
| Телеметрия | Текущие показания датчиков в реальном времени |

## Перед началом настройки

### Калибровка IMU (обязательно)

Перед настройкой коэффициентов необходимо откалибровать IMU:

1. Поставьте машинку на ровную поверхность.
2. Не трогайте её.
3. В панели **Калибровка IMU** нажмите **Полная**.
4. Подождите 3-5 секунд, пока статус не покажет **Завершена**.

### Загрузка текущей конфигурации

В панели **Стабилизация** нажмите **Загрузить с устройства** — все поля заполнятся текущими значениями с машинки.

## Выбор режима

В верхней части панели **Стабилизация** расположены кнопки режимов:

| Режим | Описание | Когда использовать |
|-------|----------|--------------------|
| **Normal** | Умеренные коэффициенты PID, базовая стабилизация | Обычная езда, начало настройки |
| **Sport** | Агрессивные коэффициенты, быстрый отклик | Быстрая езда, опытные пилоты |
| **Drift** | Мягкий контроль рыскания, контроль угла заноса | Управляемый дрифт |
| **Kids** | Повышенная стабильность, все защиты активны | Для детей |
| **Direct Law** | Стабилизация полностью отключена | Диагностика, прямое управление |

При смене режима кнопкой применяются **пресетные значения** всех коэффициентов для этого режима. Чтобы они записались на устройство, нажмите **Сохранить на устройстве**.

## Настройка параметров

### 1. Включение стабилизации

Поставьте галочку **Стабилизация включена**. Без неё PID-регулятор не влияет на управление, даже если коэффициенты заданы.

### 2. Yaw PID (контроль рыскания)

Это главный блок стабилизации — PID-регулятор, который корректирует руль для поддержания заданной угловой скорости.

| Поле | Параметр | Что делает |
|------|----------|------------|
| **Kp** | Пропорциональный | Основная реакция на ошибку. Больше = сильнее коррекция |
| **Ki** | Интегральный | Устраняет постоянное отклонение. Обычно 0 или очень маленький |
| **Kd** | Дифференциальный | Демпфирование колебаний. Гасит раскачку |
| **MaxCorr** | Макс. коррекция | Ограничивает влияние PID на руль (0.0–1.0) |

#### Порядок настройки Yaw PID

**Шаг 1 — Начните с Kp:**
- Установите `Ki = 0`, `Kd = 0`, `MaxCorr = 0.3`.
- Начните с `Kp = 0.05`.
- Катайте машинку, наблюдая за поведением при поворотах.
- Увеличивайте `Kp` на 0.02–0.05, пока машинка не начнёт ощутимо сопротивляться заносу.
- Если начались колебания (руль «виляет» туда-сюда) — уменьшите `Kp`.

**Шаг 2 — Добавьте Kd:**
- Начните с `Kd = 0.003`.
- `Kd` гасит колебания, вызванные `Kp`.
- Увеличивайте `Kd`, если машинка «виляет». Уменьшайте, если стала «вялой».
- Типичные значения: 0.002–0.015.

**Шаг 3 — Ki (опционально):**
- Если машинку постоянно уводит в одну сторону, попробуйте `Ki = 0.005–0.01`.
- Большой `Ki` вызывает раскачку с нарастающей амплитудой — будьте осторожны.

**Шаг 4 — MaxCorr:**
- Ограничивает максимальное вмешательство PID в руль.
- `0.3` — умеренное (PID может скорректировать руль на ±30% от полного хода).
- `0.5` — агрессивное.
- `0.1` — мягкое, едва заметное.

#### Типичные значения по режимам

| Параметр | Normal | Sport | Drift | Kids |
|----------|--------|-------|-------|------|
| Kp | 0.10 | 0.20 | 0.05 | 0.15 |
| Ki | 0.00 | 0.01 | 0.00 | 0.005 |
| Kd | 0.005 | 0.010 | 0.002 | 0.008 |
| MaxCorr | 0.30 | 0.40 | 0.20 | 0.35 |

### 3. Адаптивный PID

Масштабирует выход PID в зависимости от скорости машинки.

| Поле | Что делает |
|------|------------|
| **Галочка** | Включает/выключает адаптивное масштабирование |
| **SpeedRef (m/s)** | Опорная скорость — при ней PID работает как есть (множитель = 1.0) |

- На малых скоростях PID ослабляется (чтобы не дёргать руль на месте).
- На высоких скоростях PID усиливается (нужна быстрая коррекция).
- Рекомендация: включайте для режима Kids, `SpeedRef = 1.0–1.5 м/с`.

### 4. Pitch Compensation (компенсация наклона)

Автоматически корректирует газ на подъёмах и спусках.

| Поле | Что делает |
|------|------------|
| **Галочка** | Включить/выключить |
| **Gain** | Коэффициент усиления (сколько газа добавлять на градус наклона) |
| **MaxCorr** | Максимальная коррекция газа |

- `Gain = 0.01` — мягкая компенсация (1% газа на градус).
- `Gain = 0.02` — умеренная.
- `MaxCorr = 0.25` — ограничивает коррекцию 25% газа.

Полезно при езде по неровной поверхности или наклонным поверхностям.

### 5. Oversteer Warn (предупреждение о заносе)

Обнаруживает занос и автоматически снижает газ.

| Поле | Что делает |
|------|------------|
| **Галочка** | Включить/выключить |
| **SlipThresh (°)** | Порог угла скольжения для срабатывания |
| **RateThresh (°/s)** | Порог скорости нарастания скольжения |
| **ThrottleRed** | Снижение газа при заносе (0.0–1.0) |

- `SlipThresh = 20°` — срабатывает при значительном заносе.
- `SlipThresh = 10°` — срабатывает раньше (безопаснее для детей).
- `ThrottleRed = 0.5` — снижает газ вдвое при заносе.
- `ThrottleRed = 0.7` — снижает до 30% (агрессивная защита).

### 6. Детский режим

Кнопка **Детский режим: ВЫКЛ/ВКЛ** включает ограничения газа и руля на стороне веб-интерфейса:

| Слайдер | Что делает |
|---------|-----------|
| **Макс. газ (%)** | Ограничивает максимальный газ (по умолчанию 50%) |
| **Макс. руль (%)** | Ограничивает максимальный поворот руля (по умолчанию 70%) |

Это клиентское ограничение — слайдеры управления не позволят отправить значение больше заданного лимита.

Для полноценного детского режима с серверной стороны (slew rate, anti-spin) — выберите режим **Kids** кнопкой.

## Сохранение и загрузка

| Кнопка | Действие |
|--------|----------|
| **Загрузить с устройства** | Считывает текущую конфигурацию из NVS |
| **Сохранить на устройстве** | Записывает текущие значения из формы в NVS (переживает перезагрузку) |

После изменения любых параметров нажмите **Сохранить на устройстве**. Появится сообщение «Сохранено» (зелёный) или «Ошибка сохранения» (красный).

## Пошаговый сценарий настройки с нуля

1. Подключитесь к машинке по Wi-Fi и откройте `http://192.168.4.1/`.
2. Выполните **полную калибровку IMU**.
3. Нажмите **Загрузить с устройства**.
4. Выберите режим **Normal**.
5. Поставьте галочку **Стабилизация включена**.
6. Установите начальные коэффициенты:
   - `Kp = 0.05`, `Ki = 0`, `Kd = 0`, `MaxCorr = 0.3`
7. Нажмите **Сохранить на устройстве**.
8. Покатайте машинку и наблюдайте за поведением.
9. Постепенно увеличивайте `Kp` (шаг 0.02–0.05), каждый раз сохраняя и тестируя.
10. Когда найдёте оптимальный `Kp` — добавьте `Kd` (начните с 0.003).
11. При необходимости включите **Pitch compensation** и **Oversteer warn**.
12. Для проверки используйте панель **Телеметрия** — она в реальном времени показывает показания IMU, EKF и текущие значения газа/руля.

## Мониторинг в реальном времени

### Телеметрия (нижняя панель)

В реальном времени отображаются:
- **Accel X/Y/Z** — ускорения (g)
- **Gyro X/Y/Z** — угловые скорости (°/s)
- **Throttle / Steering** — текущие команды после стабилизации
- **EKF Slip** — текущий угол скольжения (°)
- **EKF Speed** — текущая скорость (m/s)
- **EKF Vx/Vy** — продольная и поперечная скорости

### Индикатор заноса

Если включён **Oversteer warn**, в панели лога появится индикатор:
- **OK** (зелёный) — всё в порядке
- **ЗАНОС!** (красный) — обнаружен занос, газ снижается

### Лог телеметрии

Для анализа после заезда:
1. Нажмите **Статистика** — покажет количество записанных кадров.
2. Нажмите **Скачать CSV (500)** — скачает последние 500 кадров.
3. Откройте CSV в Excel/Google Sheets и постройте графики `slip_deg`, `speed_ms`, `steering` для анализа.

## Troubleshooting

| Симптом | Причина | Решение |
|---------|---------|---------|
| Машинка «виляет» рулём | `Kp` слишком большой | Уменьшите `Kp` на 20–30% |
| Колебания с нарастающей амплитудой | `Ki` слишком большой | Уменьшите `Ki` или обнулите |
| Стабилизация не ощущается | `Kp` слишком маленький или стабилизация выключена | Проверьте галочку, увеличьте `Kp` |
| Руль дёргается на месте | `Kp` большой при нулевой скорости | Включите адаптивный PID |
| Машинку всё равно заносит | `MaxCorr` слишком маленький | Увеличьте `MaxCorr` до 0.4–0.5 |
| Параметры сбрасываются после перезагрузки | Не нажали «Сохранить на устройстве» | Всегда сохраняйте после настройки |
| Медленная реакция на повороты | `lpf_cutoff_hz` слишком низкий | Увеличьте через WebSocket API (см. ниже) |

## Продвинутая настройка через WebSocket

Некоторые параметры (Madgwick beta, LPF cutoff) недоступны в графическом интерфейсе, но доступны через WebSocket API. Можно использовать консоль браузера (F12 → Console):

```javascript
// Посмотреть текущую конфигурацию
ws.send(JSON.stringify({type: 'get_stab_config'}));

// Изменить Madgwick beta (чувствительность фильтра ориентации)
ws.send(JSON.stringify({
    type: 'set_stab_config',
    madgwick_beta: 0.12  // по умолчанию 0.1, диапазон 0.01–1.0
}));

// Изменить частоту среза LPF (фильтр гироскопа)
ws.send(JSON.stringify({
    type: 'set_stab_config',
    lpf_cutoff_hz: 25.0  // по умолчанию 30 Hz, диапазон 5–100 Hz
}));
```

### Madgwick beta

- **Больше** (0.15–0.3) → быстрее реакция на изменение ориентации, больше шум.
- **Меньше** (0.03–0.08) → стабильнее ориентация, медленнее реакция.
- **Рекомендация**: начните с 0.1. При сильных вибрациях уменьшите до 0.05.

### LPF cutoff (частота среза фильтра Баттерворта)

- **Ниже** (15–25 Hz) → сильнее фильтрация шума гироскопа, но больше задержка.
- **Выше** (35–60 Hz) → меньше задержка, но больше шум от вибраций.
- **Рекомендация**: 20–40 Hz для типичных RC-машин. Если моторы сильно вибрируют — снижайте.
